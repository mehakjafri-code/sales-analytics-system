# Sales Analytics System

A Python-based system to clean, validate, analyze, and report sales data.

## Features
- Cleans messy sales transaction data
- Validates records based on business rules
- Analyzes revenue by region and product
- Fetches real-time USD conversion using an external API
- Generates business reports

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the system:
python main.py

3. Output will be saved in:
output/summary.txt
